# OctoAcme Role Review Checklist

Use this checklist to regularly review and clarify roles, responsibilities, and cross-functional interactions in project teams.

## Checklist
- [ ] Roles and personas are well-defined and documented
- [ ] Responsibilities are clear for each persona (including new additions)
- [ ] Cross-team interactions are described and agreed upon
- [ ] Stakeholders have reviewed recent process documentation updates
- [ ] Accountability gaps have been addressed for each project phase
- [ ] Documentation has been updated for onboarding and reference purposes
